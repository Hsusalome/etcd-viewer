Demo about content

ETCD Viewer uses the ETCD v2.0 Rest API to communicate.

The application is built on top of the following:

  • Apache Wicket 6.19.0
  • Google Guice 3.0
  • Apache CXF 3.0.1
  • Jackson JSON 2.5.3
  • JAX RS API 2.0
  • JAVAX Inject 1.0
  • Twitter bootstrap v3.3.4
  • Font Awesome 4.3.0
  • JQuery 1.11.2
